# Privacy Policy

**Test Architect** — Last updated: May 2026

## Summary

This plugin does not collect, transmit, or store any personal data or telemetry.

## What this plugin does

Test Architect runs entirely locally inside Claude Code. It reads your project's
source and test files, and writes new test files into your local project. It has
no backend, no network calls, and no external services.

## What this plugin does NOT do

- Does not send any data to external servers
- Does not collect analytics or telemetry
- Does not transmit file contents outside your local machine
- Does not require authentication or account creation
